1. Website Usage

FUNNELWEB is a modern e-commerce demo project built using Django. Users can explore categories, search products, view product details, and experience a professional checkout flow.

2. User Accounts & Login

Users can browse products without logging in. However, placing orders, accessing the cart, and checkout requires login.

3. Orders & Checkout

During checkout, users must provide correct shipping details. Incorrect details may cause delivery failure in a real system.
