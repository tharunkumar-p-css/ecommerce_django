How FUNNELWEB Works

Flow
1) Browse Products

Users can explore products, view offers, and filter categories without login.

2) Login to Purchase

When the user tries to buy, the system ensures login for secure order tracking.

3) Cart & Checkout

Add items to cart, update quantities, and proceed to premium checkout with payment options.

4) Orders Dashboard

Users can view order history, returns/exchange flow, and order status updates.

FUNNELWEB is a portfolio demo project built using Django, HTML, CSS, and JavaScript.
